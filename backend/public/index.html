API da Barbearia do Além

Bem-vindo à API da Barbearia do Além. Esta API fornece endpoints para gerenciar agendamentos, clientes e serviços.

Endpoints Disponíveis:

Serviços

GET /api/servicos - Listar todos os serviços disponíveis

Clientes

POST /api/clientes - Criar um novo cliente

Agendamentos

GET /api/agendamentos - Listar todos os agendamentos

POST /api/agendamentos - Criar um novo agendamento

PATCH /api/agendamentos/:id - Atualizar status de um agendamento

Datas Bloqueadas

POST /api/datas-bloqueadas - Bloquear uma data

DELETE /api/datas-bloqueadas/:data - Desbloquear uma data

Para acessar o frontend, visite: http://localhost:4200
